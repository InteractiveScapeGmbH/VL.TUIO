# VL.TUIO

Tuio Library for vvvv gamma, based on [Tuio.Net](https://github.com/InteractiveScapeGmbH/TuioNet).

Supports:
- Tuio 1.1
- Tuio 2.0
- Scape X Mobile

over:
- UDP
- Websocket
